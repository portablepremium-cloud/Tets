# Snake Game

A classic Snake game implemented in Python using Pygame.

## Features

- Classic snake gameplay mechanics
- Score tracking
- Game over screen with restart functionality
- Smooth controls using arrow keys
- Modern UI with clean graphics

## Installation

1. Make sure you have Python 3.6+ installed
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Play

1. Run the game:
   ```bash
   python snake_game.py
   ```

2. Controls:
   - **Arrow Keys**: Move the snake
   - **R**: Restart the game (when game over)
   - **Q**: Quit the game (when game over)

## Game Rules

- Control the snake to eat the red food
- The snake grows longer with each food eaten
- Avoid hitting the walls or the snake's own body
- Try to get the highest score possible!

## Game Features

- **Score Display**: Current score shown in the top-left corner
- **Smooth Movement**: Responsive controls with collision detection
- **Game Over Screen**: Shows final score and restart options
- **Visual Design**: Clean, modern graphics with blue snake and red food
